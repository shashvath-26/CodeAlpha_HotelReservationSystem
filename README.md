## 📋 Project Overview

The Hotel Reservation System is a comprehensive Java application that allows users to manage hotel room bookings. It provides a user-friendly interface for viewing available rooms, making reservations, and cancelling bookings.

### Key Features
- **Room Management:** Display available rooms and all rooms with their details
- **Make Reservations:** Book rooms with guest information, dates, and room type selection
- **Cancel Reservations:** Cancel existing reservations by reservation ID
- **View Reservations:** View all active reservations or search by reservation ID
- **Multiple Room Types:** Standard, Deluxe, and Suite rooms with different pricing
- **Price Calculation:** Automatic total price calculation based on room type and nights
- **Reservation Tracking:** Unique reservation ID for each booking

## 🚀 How to Run

1. **Clone the repository:**
   ```
   git clone https://github.com/shashvath-26/CodeAlpha_HotelReservationSystem.git
   ```

2. **Navigate to the project directory:**
   ```
   cd CodeAlpha_HotelReservationSystem
   ```

3. **Compile the Java file:**
   ```
   javac HotelReservationSystem.java
   ```

4. **Run the program:**
   ```
   java HotelReservationSystem
   ```

---

## 🔄 Usage

When you run the program, you will see a main menu with the following options:

1. **View Available Rooms** - See all currently available rooms
2. **Make a Reservation** - Book a room by providing guest details
3. **Cancel a Reservation** - Cancel an existing booking using reservation ID
4. **View All Reservations** - Display all active reservations
5. **View Reservation by ID** - Search for a specific reservation
6. **View All Rooms** - Display all rooms and their availability status
7. **Exit** - Close the application
